How It Works

🎯

Content-Based

Analyzes genres, director, cast & plot using TF-IDF vectorization to find similar films.

Collaborative

Finds patterns in user ratings using item-item cosine similarity on the user-movie matrix.

🔀

Hybrid

Blends both approaches (60/40 weighted) for the most accurate, personalized results.
